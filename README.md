# Firebase の使い方の勉強として制作したTodoアプリ
## できること
データの追加、削除

データの追加は右下の＋ボタンから、削除は表示されたアイテムをクリックて表示されるチェックボックスで行う。

## やってみたこと
・ダークモードに対応

・FloatingActionButton を試した

・画面ごとのViewModelを用意

・Firebase とのデータのやり取りをリポジトリクラスに分割

## 今後やりたいこと
